### Who is it for?

| 🚛 Truck Drivers | 🏢 Businesses / Clients |
|---|---|
| Post your truck and its availability | Post the goods you need to transport |
| Browse goods that need moving | Browse available trucks nearby |
| Accept booking requests | Send booking requests to drivers |
| Track your listings from a dashboard | Track shipments from a dashboard |

## ✨ Key Features

- 🔐 **Secure Authentication** — JWT-based signup/login with role-based access (Driver vs. Client)
- 🚛 **Post & Browse Trucks** — Drivers list vehicle type, capacity, and route availability
- 📦 **Post & Browse Goods** — Clients list cargo type, weight, pickup, and destination
- 📋 **Booking System** — Request, accept, and manage bookings between drivers and clients
- 📊 **Role-Based Dashboards** — Separate, tailored dashboards for Drivers and Clients
- 📍 **Live Tracking Page** — Track shipment/trip status in real time
- 🌐 **Bilingual Support** — English and Hindi interface options
- 📱 **Fully Responsive UI** — Built with Tailwind CSS and Framer Motion for a smooth experience on any device
